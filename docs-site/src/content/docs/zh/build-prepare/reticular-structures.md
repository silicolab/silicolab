## 目标

使用内置 BENZ 组件构建默认 `2 x 2 x 2` **HCB (T3 core + T3 core)** 堆叠。当前默认结果
必须同时符合以下数值：

| 检查项 | 预期值 |
| --- | --- |
| 分子式 / 原子 / 已存储键 | `C96H48` / 144 / 168 |
| 晶胞长度 | 约 `15.637 x 15.637 x 7.200 Å` |
| 晶胞角 | `90 / 90 / 60°` |
| 已存储连接图 | 2 个连通分量，每层一个 |
| 层间共价键 | 0 |

只要有任何数值不符，就停止并返回下方固定设置。只看到双层结构或六角晶胞不能替代这些检查。

## 设置固定草稿

从干净的可写项目开始。打开 **Launch**（启动），展开 **Structure Builder**（结构构建），
选择 **Reticular Builder**（网状结构构建器）。明确输入或确认每个数值：

| 设置 | 数值 |
| --- | --- |
| Name | `Structure` |
| Network | `HCB (T3 core + T3 core)` |
| Core A / Core B | `BENZ / BENZ` |
| Linkers | `0 in chain`；不添加 linker |
| Orientational modulation | Core A 与 Core B 均关闭 |
| Functionalization | 关闭 |
| Preview supercell | `2 x 2 x 2` |
| Layer spacing | `3.6 Å` |
| 自定义构建块 | `0 custom blocks loaded`；本工作流不要导入自定义构建块 |

两个 BENZ 组件都是 T3 core，各有三个连接位点。本工作流不需要输入文件。

## 预览并构建

1. 选择 **Preview**（预览）并检查草稿。预览会改变视口，但不会生成已接受的结果条目。
2. 再次核对所有固定设置。草稿有变动时，修正后重新预览。
3. 只有设置一致时才选择 **Build**（构建）。SilicoLab 创建并激活名为 `Structure` 的新条目；
   **Activity**（活动）把 **Reticular Structure Builder** 记录为 `Completed`。
4. 在 Details（详细信息）和结构摘要中检查当前条目，再检查其已存储连接图。只有分子式、计数、
   晶胞、分量和层间连接全部符合上方完整检查点时才继续。

`c` 长度为 `2 x 3.6 Å = 7.200 Å`，对应两个层重复。因此，此处应有两个连通分量，
并不表示构建失败。

## 键、电荷与文件的含义

- 每个 BENZ 组件内原有的键类型会保留；不同 core 之间的新连接为 single bond（单键）。
- 新连接的原子对按 UFF equilibrium bond length（UFF 平衡键长）放置。这只是初始坐标的
  放置规则，不是 UFF 能量最小化，也不是完整的力场模型。
- 每个输出原子的已存储电荷都设为 `0.0`；组件电荷不会带入结果。该数值不是经过验证的
  部分电荷或氧化态。
- 两层之间相隔 `3.6 Å`，且没有层间共价键。已存储连接图不表示范德华或静电相互作用，
  因此堆叠本身不能说明层间结合或稳定性。
- 条目会获得建议的相对路径 `Structure.cif`，但 Build 不会自动写出 CIF。需要文件时，
  使用导出工作流并选择目标位置。

## 恢复到已知状态

| 现象 | 恢复方法 | 继续前的检查 |
| --- | --- | --- |
| 未选择 BENZ/BENZ，或启用了 linker、取向选项或 functionalization | 恢复每个固定值并重新选择 **Preview** | HCB T3/T3、BENZ/BENZ、无 linker、两个取向选项关闭且 functionalization 关闭 |
| 无法确定预览状态 | 在 Build 前选择 **Cancel**（取消），重新打开构建器并恢复固定草稿 | 尚未接受任何结果条目 |
| 使用错误设置执行了 Build | 在干净的可写项目中重新开始；不要从错误结果继续 | 新条目符合完整定量检查点 |
| 任一分子式、计数、晶胞值或连通性数值不符 | 停止；核对所有设置后重新构建 | 只有匹配的 `2 x 2 x 2` 结果属于本工作流 |
| 磁盘上没有 `Structure.cif` | 保持已核对的结果为当前条目并显式导出 | 确认所选格式和目标位置 |

只要任何结构检查点仍不确定，就不要继续编辑、导出或计算。

## 科学限制

本工作流只覆盖内置 HCB T3/T3 网络。自定义 SLF 组件不在本页范围内；不要使用来源不可信
或未经验证的组件元数据。

构建只组装模板和初始坐标；它不会恢复组件电荷、生成经过验证的电荷模型或完整力场参数、
最小化或平衡结构，也不能说明孔可达性、机械或热力学稳定性、可合成性或材料身份。
进入后续计算或解释前，应验证拓扑、价态、电荷、周期连接、几何、参数化和所需松弛。
