## 启动图形界面

不带参数运行 SilicoLab 即可打开图形界面：

```sh
silicolab
```

在图形界面中，你可以查看和编辑 3D 结构、绘制 2D 分子草图、运行几何优化、
量子化学计算与分子动力学模拟，并对结果绘图。

## 无界面运行脚本

传入脚本路径即可在命令行中运行，无需打开图形界面：

```sh
silicolab workflow.sls
```

同样的 `.sls` 脚本也可以在图形界面的控制台中交互执行——在命令行里自动化
的工作流，在应用内的行为完全一致。
