## 调整当前结构的方向

相机操作只改变视图，不会改变原子坐标或结构内容。

| 目的 | 操作 | 可见结果 |
| --- | --- | --- |
| 旋转 | 在视口中按主键拖动 | 结构绕视图中心旋转 |
| 平移 | 按次键或中键拖动 | 整个结构在视口中移动 |
| 平移或缩放 | 在视口上使用滚轮、触控板、捏合或带修饰键的滚动 | 映射取决于输入设备；检查最终位置或比例 |
| 恢复相机 | 按 `R` 执行 **Reset View** | 相机恢复到包含结构的视图；样式和可见性保持不变 |

应以可见结果为准，不要假设每种鼠标或触控板都以完全相同的方式映射手势。

## 改样式前确认原子范围

打开 **Style**（样式），先阅读顶部的范围提示。

| 当前原子选择 | Style 动作范围 | 面板提示 |
| --- | --- | --- |
| 已选择一个或多个原子 | 当前条目中的这些原子 | `Acting on N selected atom(s)` |
| 没有选择原子 | 当前条目中的全部原子 | `No atoms selected - styles apply to all atoms` |

Entries 侧栏中的条目和分组高亮不会影响 Style 范围。若要修改整个结构，应先清除原子选择，
确认视口和 Sequence 高亮消失，再确认面板显示应用到全部原子的消息。

## 选择一种基础表示

在 **Representation**（表示方式）下选择一种互斥的基础表示。

| 选项 | 显示变化 |
| --- | --- |
| **Ball-and-stick** | 显示原子球和键柱 |
| **Stick** | 用键柱强调连接关系 |
| **Wireframe** | 用细线显示键，不显示原子球 |
| **Sphere (VdW)** | 按元素显示半径绘制较大的渲染球体 |
| **Dots** | 把原子显示为小点 |

选择表示方式后，应同时检查可见几何和临时状态反馈中的原子数。基础表示只改变渲染，不会
修改元素、坐标、化学键或其他结构数据。

## 添加 Cartoon 或 Surface 叠加层

**Cartoon ribbon** 和 **Surface** 是附加叠加层，可以同时存在，也不会替换当前基础表示。
它们的开关使用与其他 Style 动作相同的原子范围。

- Cartoon 需要兼容的链、残基和主链元数据。没有可见带状结构，本身不表示导入失败，也不
  证明结构中没有聚合物。
- Surface 是根据原子位置和显示半径生成的渲染几何。它不是电子密度、静电势、疏水性或
  其他计算得到的化学表面。

第一次启用 Surface 时可能需要等待几何生成。**Advanced > Surface** 下的外观控件会调整
当前条目表面的填充或网格样式和透明度。

关闭叠加层会从当前原子范围移除该层，同时保留基础表示。

## 显示、隐藏或隔离原子

可见性独立于基础表示和叠加层。

| 动作 | 结果 |
| --- | --- |
| **Show** | 显示当前样式范围中的原子 |
| **Hide** | 隐藏当前样式范围中的原子 |
| **Isolate** | 只显示当前范围，并隐藏当前条目中的其他原子 |

隐藏原子不会删除它们。使用 Isolate 后若要恢复整个结构，应先清除原子选择，确认没有原子
仍被选中，然后选择 **Show** 或 **Reset to default style**。否则该动作仍然只处理已选原子，
其他原子会继续隐藏。

**Reset to default style** 会重置当前样式范围的表示方式、Cartoon 和 Surface 叠加状态以及
可见性，但不会重置相机。**Reset View** 只重置相机，不会恢复样式或隐藏原子。

## 区分原子样式与条目级显示设置

即使只选择了部分原子，以下控件仍作用于当前条目的整个视口：

- **Background**、**Unit cell** 和 **Atom labels**；
- **Light**、**Silhouettes** 和轮廓宽度；
- Cartoon 的截面和平滑设置；
- Surface 的填充或网格外观与透明度。

Surface 开关决定哪些原子参与该叠加层；Surface 外观控件决定当前条目的表面如何绘制。
每个条目可以保留自己的视图状态，因此切换条目后应重新检查。

## 不要从显示样式推断化学性质

元素颜色、链颜色、选择高亮、表示方式、可见性和渲染表面都只是显示选择。除非有另一个
明确标注的分析结果提供证据，否则不要把它们解释为电荷、静电势、疏水性、键级置信度、
可及性或其他化学性质。
